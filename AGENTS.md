# A 股回测项目协作约定

## 默认工作区 Skill

- 本工作区默认同时遵循 `.codex/skills/china-quant-backtest/SKILL.md`。
- 如果当前会话没有把 `china-quant-backtest` 自动注册为内置 skill，也按工作区内这份本地 skill 文档手工执行，不需要用户重复说明。
- 如果 `.codex/skills/china-quant-backtest/SKILL.md` 与本文件冲突，以 `AGENTS.md` 为准。

## 默认上下文

- 处理 Tushare 数据时，默认从环境变量 `TUSHARE_TOKEN` 读取 token；不要把 token 明文写入代码、文档或提交记录。
- 默认按 2000 积分权限规划数据拉取。如果某个接口是否可用不确定，先明确写出假设，再继续实现。
- 除非用户明确要求英文，项目中的数据说明、指标说明、README 更新都使用中文。

## GitHub 交付与推送

- 默认 GitHub 仓库地址为 `https://github.com/XigZhou/T_0_system`。
- 默认分支为 `master`。
- Git 提交作者默认使用本机已配置身份；当前约定的联系邮箱为 `xigzhous@gmail.com`。
- 提交信息默认使用中文。
- 当本次改动已经完成并且本地验证通过后，默认继续执行 `git add`、`git commit`、`git push`，不需要再次单独确认。
- 优先使用 SSH 方式连接 GitHub，不使用密码登录；如果 SSH 未授权，先提示补充公钥授权，再继续推送。
- 如果当前目录还不是 Git 仓库，则先初始化 Git 仓库、绑定远程仓库 `origin`，再按默认分支推进。
- 如果远程仓库已有历史且与本地初始化历史不一致，先明确说明情况，再选择合并、拉取重放或其他安全方案，不直接强推覆盖。

## 数据与指标文档

- 与项目数据相关的说明文档放在仓库根目录下的 `docs/`，与 `README.md` 同级。
- 新增或修改数据集时，同步补充中文说明，至少包含：来源接口、输出文件路径、数据粒度、主键字段、更新时间、字段解释、缺失值处理、复权或停牌处理逻辑。
- 新增或修改指标时，同步补充中文说明，至少包含：指标用途、输入字段、计算公式、窗口参数、边界条件、一个带数字的计算示例。
- 优先复用并维护 `docs/data-dictionary-template.md` 与 `docs/indicator-documentation-template.md`，再按具体项目生成正式文档。

## 回测系统交付

- 面向交付的回测系统默认要有前端界面；如果延续现有前端，至少补齐使用方式和启动说明。
- `README.md` 必须覆盖：准备工作、环境变量、数据准备、启动方式、核心功能入口、复现结果的方法与输出位置。
- 生成系统使用文档 `docs/system-documentation.md`。解释如何使用系统的每一个功能模块包括输入参数、输出结果、异常处理等。
- 交付前先本地验证，再对外说明完成。至少执行与本次改动直接相关的测试；如果涉及 API 或前端，还要做一次启动冒烟验证。
- 可以先运行 `python scripts/verify_delivery.py` 做基础文档检查，再补充更具体的测试和启动验证。
- 如果当前目录不是 Git 仓库，不能声称已经完成 GitHub 上传，只能说明还需要初始化仓库或切到正确目录后再推送。

## 腾讯云服务器

- 当用户提到腾讯云服务器或远程主机时，优先使用本机已配置的 SSH 别名或既有连接方式操作。
- 如果目标主机不明确，再向用户确认主机别名、用户名或目标目录；不要把密码、私钥或敏感连接串写入仓库。

## 股票动量计算公式
- 20日的股票动量计算公式为 T日close价格 - (T-20+1)日close价格 然后除以(T-20+1)日close价格，比如五日动量计算
m5=[T日close价格-(T-4日close价格)]/(T-4日close价格)， 以此类推

## 探索回测系统最适合的买入条件
- 初始资金10万元，滑点和买入卖出都要手续费，手续费为0.003%，没有最低消费
- 每次买入每只股票的总价格为一万元左右，买入股票数必须为100或其整数倍（如：股价20元买500股、股价30元买300股、股价120元买100股）。买入和卖出都要加上滑点和手续费，手续费为0.003%，没有最低消费
- 每次系统帮忙探索出合适的买入条件，需要生成两个文档，一个是探索的买入条件是怎么样，然后不同的买入条件的回测结果是什么。另外一个是，这次探索里面的买卖记录，需要有每笔交易的日期，买入，卖出，股票代码，股票名称，股票价格，股票数量，手续费，总金额，如果是卖出单还需要有盈亏价格计算，方便我校验
- 信号指标的计算用前复权价格，买入和卖出用除权价格
