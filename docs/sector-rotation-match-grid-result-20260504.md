## 5. TopN 改变情况

入选记录显示，本轮最优的 `主线簇匹配加权_w5` 与原 `板块候选_score0.4_rank0.7` 的 TopN 重合率为 `95.57%`，只改变了 `34` 个入选记录，但收益从 `87.29%` 提升到 `91.61%`。

这说明：轮动不是不能配合 TopN，而是不能使用“市场级常数”直接加分。使用股票差异化字段后，轻微重排 TopN 是有效的。

各策略入选股票的平均匹配率：

| 策略 | 股票匹配主线簇均值 | 股票匹配 Top 主题均值 |
| --- | ---: | ---: |
| 板块候选_score0.4_rank0.7 | 70.20% | 44.83% |
| 主线簇匹配加权_w5 | 72.52% | 45.54% |
| 主线簇匹配加权_w10 | 73.27% | 46.04% |
| Top主题匹配加权_w8 | 71.78% | 47.77% |
| 候选_股票匹配主线簇 | 100.00% | 64.33% |
| 候选_股票匹配Top主题 | 100.00% | 100.00% |

硬过滤虽然让匹配率变成 100%，但收益明显下降，说明“必须匹配主线”太严格，会过滤掉有效候选。

## 7. 结论

1. 不应该去掉 TopN。TopN 仍然负责资金分配和候选排序。
2. 市场级轮动字段不能直接加到评分里，因为同日所有股票获得同一常数，不改变排序。
3. 股票差异化轮动字段可以配合 TopN。`stock_matches_rotation_top_cluster` 小权重加分有效。
4. 当前最好方案是 `主线簇匹配加权_w5`：收益 `91.61%`，超过原板块候选 `87.29%`，但回撤从 `11.30%` 升至 `11.75%`。
5. 硬要求股票匹配主线簇或 Top 主题效果不好，收益分别只有 `62.23%` 和 `65.45%`。
6. Top 主题匹配加权不如主线簇匹配加权，说明主题簇层面比具体 Top1 主题更稳。
7. 新主线启动惩罚效果不好，`p5/p8` 都明显低于原板块候选。

## 8. 下一步建议

下一步建议围绕 `主线簇匹配加权_w5` 做稳定性验证，而不是继续扩大权重：

1. 对 `基准动量`、`板块候选_score0.4_rank0.7`、`主线簇匹配加权_w5`、`候选_避开新能源主线` 做分年度、最近一年和滚动窗口对比。
2. 若 `主线簇匹配加权_w5` 在 2023、2024、2025、2026YTD 都不劣化，再考虑生成第三个模拟账户观察。
3. 不建议接入硬过滤版本，也不建议接入新主线启动惩罚版本。
4. 后续可以在 `w5` 附近做细网格，例如 `2,3,4,5,6,8`，确认最佳权重不是偶然值。
