## 5. 关键结论

1. 轮动硬过滤没有超过原板块候选。`板块候选_score0.4_rank0.7` 仍然是本轮最高收益组合，账户收益 `87.29%`。
2. `候选_避开新能源主线` 是更稳一点的备选：收益 `79.40%`，超过基准 `75.86%`，回撤略低于基准，信号中位收益为正。但它仍低于原板块候选。
3. `候选_新主线启动` 表现不佳，收益 `60.11%`，胜率 `46.67%`，信号中位收益 `-0.67%`。这验证了上一轮诊断的判断：直接追新主线不适合作为硬过滤。
4. `候选_退潮或观察`、`候选_主线退潮`、`候选_轮动观察` 虽然在交易打标统计里看起来改善单笔质量，但作为硬过滤会大幅减少交易机会，账户收益明显下降。
5. `候选_科技成长且非新启动` 的胜率和中位收益较好，但只有 42 次买入，样本偏少，只能作为后续小样本观察。

## 6. 当前策略判断

本次实验不支持把轮动状态作为硬买入过滤直接接入主策略。

更合适的处理方式是：

- 继续保留 `板块候选_score0.4_rank0.7` 为候选增强策略；
- 把 `候选_避开新能源主线` 作为保守备选策略，而不是替代原候选；
- 暂时不要接入 `新主线启动` 硬过滤；
- 后续如果使用轮动，应优先尝试“评分加权”或“风险提示”，而不是直接过滤掉大部分交易日。

## 7. 下一步建议

下一步建议做两件事：

1. 对 `板块候选_score0.4_rank0.7`、`候选_避开新能源主线`、`基准动量` 做分年度和最近一年对比，判断收益是否集中在某一年。
2. 如果要继续研究轮动，不再做简单硬过滤，而是做轮动加权评分，例如：

```text
基础评分
+ rotation_top_cluster_tech * 轻微加分
- rotation_top_cluster_new_energy * 轻微扣分
- rotation_is_new_start * 轻微扣分
```

这样可以保留交易机会，同时让轮动状态影响排序，而不是直接把信号砍掉。
